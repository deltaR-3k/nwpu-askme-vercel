# 快速开始指南

## 一键启动（Windows）

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **配置API Key**
   
   创建 `.env` 文件，添加：
   ```
   OPENAI_API_KEY=你的OpenAI_API_Key
   ```

3. **启动服务**
   
   双击运行 `start_server.bat`，或在命令行执行：
   ```bash
   python rag_backend.py
   ```

4. **访问系统**
   
   打开浏览器访问：http://localhost:5000

## 功能说明

### 🔍 搜索文档
- 输入关键词，点击"搜索文档"
- 系统返回相关文档列表，按相似度排序
- 点击文档可展开查看完整内容

### 💬 智能问答
- 输入问题，点击"智能问答"
- 系统基于相关文档生成详细回答
- 同时显示答案和相关文档来源

### 📊 调整参数
- 通过"返回文档数"选项调整检索数量（1-20）
- 更多文档可以提供更全面的答案，但响应时间可能更长

## 注意事项

⚠️ **首次运行**：
- 系统会使用OpenAI API为所有文档生成嵌入向量
- 需要有效的OpenAI API Key和网络连接
- 生成文档嵌入需要调用API，会产生费用（约107条文档）
- 文档嵌入会在启动时生成并缓存在内存中，后续启动无需重新生成（除非数据更新）

⚠️ **API费用**：
- 智能问答功能会调用OpenAI API，会产生费用
- 建议使用GPT-3.5-turbo模型（已配置），成本较低

⚠️ **数据更新**：
- 如需更新数据，运行 `python merge_data.py` 重新合并
- 合并后需要重启后端服务

## 常见问题

**Q: 提示"无法连接到后端服务器"？**
A: 确保后端服务已启动（运行 `python rag_backend.py`）

**Q: 提示"请设置OPENAI_API_KEY"？**
A: 创建 `.env` 文件并添加你的API Key

**Q: 搜索没有结果？**
A: 尝试使用不同的关键词，或增加返回文档数量

**Q: 答案不够准确？**
A: 尝试使用更具体的问题描述，或调整返回文档数量
